**Admin User Stories**

1. As an Admin, I want to create and manage doctor accounts so that doctors can securely access the system.

2. As an Admin, I want to create and manage patient accounts so that patients can book and track appointments.

3. As an Admin, I want to assign roles and permissions so that users only access features relevant to their role.

4. As an Admin, I want to view all appointments in the system so that I can monitor clinic activity.

5. As an Admin, I want to enable or disable user accounts so that I can maintain system security and control access.

**Patient User Stories**

1. As a Patient, I want to register and log into the portal so that I can access appointment services.

2. As a Patient, I want to view available doctors and time slots so that I can book an appointment that fits my schedule.

3. As a Patient, I want to book an appointment online so that I do not need to call the clinic.

4. As a Patient, I want to view my upcoming and past appointments so that I can track my visits.

5. As a Patient, I want to cancel or reschedule appointments so that I can adjust plans when necessary.

**Doctor User Stories**

1. As a Doctor, I want to log into the system so that I can manage my appointments securely.

2. As a Doctor, I want to define my availability so that patients can only book appointments during valid time slots.

3. As a Doctor, I want to view my daily and upcoming appointments so that I can plan my schedule efficiently.

4. As a Doctor, I want to access patient appointment details so that I am prepared for consultations.

5. As a Doctor, I want to update appointment statuses (e.g., completed or cancelled) so that records remain accurate.
